นาฬิกาชีวภาพประสาน 3 สิ่ง

Sleep-wake cycle

เวลาหลับ–ตื่นที่สม่ำเสมอเป็นแกนกลาง

Melatonin rhythm

เพิ่มในช่วงมืด ถูกกดได้จากแสงกลางคืน โดยเฉพาะหน้าจอสว่าง

Body temperature rhythm

อุณหภูมิร่างกายมีจังหวะขึ้นลงสัมพันธ์กับความง่วงและความตื่นตัว

⏱️

1) Sleep duration

ระยะเวลาการนอนเพียงพอ ไม่สั้นเกินไปหรือยาวผิดปกติเมื่อเทียบกับบริบทผู้ป่วย

🏗️

2) Sleep architecture

โครงสร้างการนอนดี มีช่วงหลับลึกและการนอนที่ต่อเนื่องเพียงพอ

🧭

3) Circadian alignment

เวลานอนสอดคล้องกับชีวิตประจำวัน งาน ครอบครัว แสง และกิจกรรมกลางวัน

Morning light

Phase advance

แสงเช้าช่วยเลื่อนนาฬิกาชีวภาพให้เร็วขึ้น → ง่วงเร็วขึ้น ตื่นเร็วขึ้น เหมาะกับคนที่นอนดึกตื่นสาย

ใช้คู่กับ: ตื่นเวลาเดิมทุกวัน, รับแสงเช้า, ลดงีบยาว, ลดหน้าจอดึก

Evening / night light

Phase delay

แสงเย็นหรือแสงกลางคืน โดยเฉพาะหน้าจอสว่าง อาจเลื่อนนาฬิกาให้ช้าลง → นอนดึกขึ้น ตื่นสายขึ้น

ลดความสว่างจอ / ใช้ night mode / หลีกเลี่ยง bright light ก่อนนอน / ทำ ritual เข้านอน

1) Delayed sleep-wake phase disorder

“นอนไม่หลับ นอนตี 2–3 ตื่นสาย ถ้าได้นอนตื่นสายก็ไม่มีปัญหา” มักพบในวัยรุ่นหรือ night owl

แนวทาง: แสงเช้า + ลดแสงเย็น/ดึก + อาจพิจารณา timing ของ melatonin ตามบริบท

2) Advanced sleep-wake phase disorder

“ตื่นตี 4 แล้วนอนต่อไม่ได้” พบมากขึ้นในผู้สูงอายุ เป็น age-related change ได้

แนวทาง: รับแสงช่วงเย็น หรือปรับยอมรับเวลาเข้านอนที่เร็วขึ้นตามนาฬิกาชีวิต

3) Irregular sleep-wake rhythm disorder

หลับเป็นช่วง ๆ fragmented sleep กลางคืนไม่นอน กลางวันง่วง พบได้ใน dementia

แนวทาง: จัดตารางกิจกรรมกลางวัน แสงกลางวัน routine และ caregiver support

4) Non-24-hour sleep-wake rhythm

นาฬิกาชีวภาพเคลื่อนไปข้างหน้าเรื่อย ๆ มักพบในคนตาบอดที่ไม่มีแสงช่วยตั้งนาฬิกา

แนวทาง: ต้องประเมิน pattern ต่อเนื่องและจัดการเฉพาะราย

อายุที่มากขึ้นทำให้ Circadian rhythm แย่ลงอย่างไร?

🧠

Central clock เปลี่ยน

SCN neuron ลดลง การสั่งการลดลง และ melatonin rhythm เปลี่ยน

👁️

Light input ลดลง

จอประสาทตาตอบสนองลดลง ต้อกระจกหรือ lens yellowing ทำให้รับแสงลดลง

🚶

Zeitgeber ทางสังคมลดลง

ออกจากบ้านน้อย กิจกรรมลดลง นอนกลางวันมาก และ social interaction ลดลง

ประเด็นสำคัญ: sleep-circadian disturbance ใน dementia เพิ่มภาระผู้ดูแล เช่น nighttime wandering และ fragmented sleep ควร detect ให้ได้ก่อนให้ยานอนหลับ

Bedside Toolkit

  • ✅ Sleep diary / sleep-wake log
  • ✅ Caregiver report โดยเฉพาะผู้สูงอายุหรือ dementia
  • ✅ Actigraphy เมื่อมีข้อบ่งชี้หรือ pattern ไม่ชัด
  • ✅ Bright light therapy และ melatonin timing ตามปัญหา

Key Message

Prescribe a routine before a hypnotic

ลอง sleep log, ตั้งเวลารับแสง, วางกิจวัตรกลางวัน–กลางคืน และดู pattern “เมื่อไร” ก่อนรีบสรุปว่าเป็น insomnia และให้ยานอนหลับทันที
